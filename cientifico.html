Team Description Paper - TDP

O Team Description Paper (TDP) é um documento exigido para classificação no campeonato e deve descrever o robô da equipe com informações de software e hardware. O TDP também precisa deixar claro os avanços da equipe entre os anos de participação, servindo como inspiração para novos competidores ou de referência para veteranos.





Para ajudar as equipes, em breve vamos disponibilizar um template em Latex que pode ser usado como base para o TDP.

É possível acessar os TDPs das equipes desde 2020 em VSS Github TDP
